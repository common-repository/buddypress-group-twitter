== Description ==

Attach Twitter accounts to a BuddyPress group then aggregate and track tweets within that group.

Group tweets will be pulled automatically every hour, or every 30 minutes if someone visits a group home page.

== Installation ==

Drop the buddypress-group-twitter folder into your wp-content/plugins directory. Activate the plugin.

To begin using the plugin, create a group. You will see a new group creation step where you can add twitter accounts. Existing groups can use the plugin by hitting the "Admin" link within the group and adding twitter accounts.
